Reading the matrix legend
done/total Stories complete (done+superseded) over total in that release × system cell. Green when fully done, orange while work remains.
READY Every not-superseded story in the release is done regardless of system — no column, including (untagged)/(unknown), has remaining work. Shippable.
lagging: … The release has remaining not-done work; the verdict names each laggard column (which may include (untagged) and (unknown)).
(untagged) Stories with system: null — never dropped; a legacy/untagged roadmap collapses to only this column.
(unknown) Stories with a non-null, undeclared system (orphaned by a manual config edit) — shown only when present so they are never dropped; counts as a laggard. Fix via system rename / set-system null.
(untiered) The row for stories with no release (active, not on a named train). Parked backlog work is not shown here — it is not a shippable release; see complete backlog and the by-release grouping.
