Critical Findings 1
CR-042-C1

Export artifact written before the transaction commits

src/export/worker.ts:212

The worker uploads the finished file to the object store and marks the export ready before the surrounding database transaction commits. If the commit fails, the user is notified of a downloadable artifact that points at orphaned state, and a retry produces a duplicate.

Suggestion Move the ready transition and notification into an after-commit hook so the artifact is only advertised once the export row is durably persisted.
Important Findings 2
CR-042-I1

Range requests ignore the If-Range header

src/export/download.ts:88

A resumed download whose artifact was regenerated mid-transfer will silently stitch bytes from two different files. Honoring If-Range against the artifact ETag avoids serving a corrupt blend.

Suggestion Validate If-Range against the stored ETag; fall back to a full 200 response when it does not match.
CR-042-I2

Idempotency key omits the export format

src/export/request.ts:54

Two requests for the same workspace that differ only in format (CSV vs JSON) collide on the idempotency key and the second returns the first's artifact.

Minor Findings 3
CR-042-M1

Magic number for expiry window

src/export/expiry.ts:9

The 72-hour expiry is hard-coded; lift it to config alongside the other retention settings.

CR-042-M2

Inconsistent log field casing

src/export/worker.ts:140

Logs mix exportId and export_id; pick one to keep downstream queries simple.

CR-042-M3

Missing JSDoc on the public request helper

src/export/request.ts:20

The exported requestExport helper has no doc comment; add one describing the returned tracking shape.

Reviewer Notes

The shape of the pipeline is right and the worker/queue split is clean. The blocking issue is C1 — advertising an artifact before the transaction commits is a correctness bug that will surface as duplicate exports under load. Please resolve it and the two Important findings; the Minor items are welcome but not required for this cycle.

Once C1 and the Important findings are addressed I expect a quick re-review.
